Welcome to PassGen!

Features

PassGen is an app used to create, store, and generate passwords. Use the navigation buttons below to use the different functionality of this app. the plus button is used to create new passwords, the three line button lists all passwords, the profile button lets you change various aspects and settings, and the magnifying glass lets you search your list of passwords.

Security

This app enables you to create, store, and generate passwords. While we did our part to make it as secure as possible, we cannot guarentee inpenetribility. Be sure to use other features of your phone like: password protecting it or using fingerprint security. Also, there is no guarentee that your passwords will not be lost. To ensure complete saftey of passwords, we implemented a feature that downloads the passwords into an external file for your safe keeping. Also, sometimes pencil and paper is the best backup you can use. Thanks for using this app. We hope it is useful and provides convienence.
